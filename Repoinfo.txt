Dependencies for MyApp:

1. React 18.2.0		
2. Axios 1.3.1
3. Material-UI 5.11.7
4. React-router-dom  6.8.0

Instructions for installation:

1. Clone the repository for MyApp:
    git clone https://github.com/skwahaj/task

2. Navigate to the project directory:
    cd task

3. Install the dependencies:
    npm install 

4. Start the development server:
    npm start
